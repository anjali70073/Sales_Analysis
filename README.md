# 📊 Sales Analysis Project

## 📌 Overview

This project analyzes sales data to uncover insights and support better business decisions using Python.

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib, Seaborn
* Jupyter Notebook

## 📂 Dataset

* Sales data containing customer, product, and purchase details

## 🔍 Key Tasks

* Data cleaning and preprocessing
* Handling missing values
* Exploratory Data Analysis (EDA)
* Data visualization

## 📈 Key Insights

* Identified customer trends based on state, age, gender, and occupation
* Discovered top-selling products and categories
* Provided insights to improve customer experience and sales strategy

## 📁 Project Files

* `Data Cleaning.ipynb` – Analysis notebook
* `Sales Data.csv` – Dataset

## 🚀 Conclusion

Demonstrates how data analysis helps understand customer behavior and improve sales performance.
